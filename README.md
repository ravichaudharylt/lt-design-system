# LambdaTest Design Token Sheet

Static, browser-viewable catalog of all `--lt-*` design tokens used across the LambdaTest web platform.

Each row shows:
- Token name
- Light + dark color value with inline preview
- Total uses across the codebase
- Breakdown of how the token is used (text / bg / border / shadow / fill / stroke / utility class / gradient / other)
- Which products consume the token

Filter by property type and/or product. Live page: see GitHub Pages URL in repo About.

Generated by `build_token_sheet.pl` from `tokens.css` + a recursive scan of the lt-web-platform monorepo and lt-components source.
